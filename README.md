# 🚚 AI-Powered Courier Delivery Time Prediction

## 📌 Project Overview

This project predicts the expected delivery time of courier shipments using a Machine Learning model trained on historical delivery data.

The application provides users with an estimated delivery time, expected delivery date, and a confidence score based on shipment details.

---

## 🎯 Project Objectives

- Predict courier delivery time accurately.
- Reduce manual estimation.
- Improve shipment planning.
- Demonstrate the use of Machine Learning in logistics.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Power BI
- Joblib

---

## 📥 Input Parameters

- 📍 Pickup Location
- 🎯 Destination
- 🚚 Courier Type
- 📊 Historical Deliveries (Days)

---

## 📤 Output

- 📅 Estimated Delivery Time
- 📆 Expected Delivery Date
- 🎯 Confidence Score
- 🚚 Delivery Status

---

## 🤖 Machine Learning Model

Algorithm Used:

**Random Forest Regressor**

---

## 📊 Dashboard

Power BI Dashboard includes:

- Delivery Overview
- Delivery Analysis
- Prediction Dashboard

---

## ▶️ How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
